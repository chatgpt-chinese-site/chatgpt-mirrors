**什么是 ChatGPT 镜像网站？**

ChatGPT镜像网站（ChatGPT Mirror）是指将原始官网的内容复制并放置在另一服务器上，用户在国内无需翻墙即可使用这些网站的服务。与官网相比，这些镜像网站更适合国内用户，具备以下优势：

- **无需翻墙**：用户可以直接在国内使用这些服务，无需依赖代理工具。
